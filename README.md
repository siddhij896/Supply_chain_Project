# Supply Chain Analytics Dashboard

## Project Overview

This project focuses on analyzing supply chain operations using Python, Excel, SQL, and Power BI. The objective is to identify insights related to sales performance, profitability, customer segments, product categories, shipping methods, and regional performance. The project demonstrates the complete data analytics workflow from data cleaning to dashboard creation.

## Objectives

* Analyze sales and profit trends.
* Evaluate product and category performance.
* Understand customer purchasing behavior.
* Examine shipping and delivery performance.
* Generate actionable business insights through data visualization.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* Microsoft Excel
* MySQL
* Power BI
* GitHub

## Project Workflow

1. Data Cleaning using Excel
2. Exploratory Data Analysis using Python
3. Business Analysis using SQL
